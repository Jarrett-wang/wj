
1：git是什么？
    Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。


2：为什么学习git？
    是为了解决多人合作时候可能出现的代码管理方面的不方便的问题.

3：如何使用git？

    安装git软件

    在码云上创建一个远程仓库

    在本地新建一个本地仓库，或者叫本地项目，使用git初始化项目
    git init

    创建用户名,油箱
    git config --global user.name 用户名
    git config --global user.email 用户油箱

    查看文件状态
    git status

    查看文件状态
    git log

    把项目推送到暂存区
    git add "文件名称" 
    git add .  //添加所有文件
 

    把项目推送到历史区
    git commit -m"注释"

    在本地回退或者切换版本
    git reset --hard "版本号"

    把项目推送到远程仓库
    git push "远程仓库的地址" master

    把远程仓库的代码拉取到本地
    git pull "远程仓库的地址" master


    在本地创建分支，切换分支dev
    git branch dev

    切换分支
    git checkout dev

    创建以及切换分支
    git checkout -b dev

    在远程仓库创建分支dev，并且从本地分支dev向远程仓库分支dev提交代码

    在远程仓库把分支dev代码合并到主分支master
